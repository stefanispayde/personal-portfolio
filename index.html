PET-FINDER

This app was originally a group project, in which I collaborated with 2 other developers. It is a full MEAN stack utilizing API from Pet-Finder, an organization that has a database of all the adoptable domestic animals in the US, and the shelters that house them. We loved the idea of being able to search by zipcode, include filters to narrow your search, and find the perfect pet for the user and their family. There were many challenges that arose both technically and logistically, but in the short time we had, we built an app that produces correct and up to date information on animals needing a home. Features include two different ways to search; the first allows the user to choose the zipcode of the area they want to search, type of animal, breed, and gender from a dropdown menu. The second option is where the user can put in a zipcode and view all the shelters affiliated with Pet-Finder, along with their contact informtion. Another feature within the app enables the user to create a wishlist of the animals they are interested in getting more information on to save and view at any time. This requires the user to register and create a user name and password. Once they do, they can view their wishlist, as well as add or delete animals up for adoption. Building this not only allowed me to hone in on new programming skills, but was also a chance to work with other developers remotely, and work as a team. I have taken the liberty of reorganizing some of the pages functions and RESTful routes, as well as embellishing on the existing styling. It is a project I am proud of, and continue to work on, as I learn new skills. Software/Languages/Frameworks used: MongoDB, Express.js, Node.js, EJS, HTML, CSS, Sessions, Angular.js, RESTful routing
